# 🚀 Getting Started

## Prerequisites

- Python 3.8+
- Jupyter Notebook
- Power BI Desktop
- pandas
- numpy
- matplotlib
- seaborn

---

## Installation

Clone the repository:

```bash
git clone https://github.com/RashiBansal19/DataScience_Project.git
```

Move into the project folder:

```bash
cd DataScience_Project
```

Install the required Python libraries:

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```
